# Netflix User Interface Clone  

This repository contains the code for a **Netflix user interface clone**, built using **HTML** and **CSS**. The project recreates the front-end design of Netflix with responsiveness and modern styling.  

## Features  

- **Dynamic Hero Section:** Highlights Netflix's promotional content.  
- **Multi-Section Layout:** Includes sections for TV compatibility, offline downloads, kid-friendly profiles, and streaming flexibility.  
- **FAQ Section:** Features an interactive design for frequently asked questions.  
- **Responsive Design:** Adapts to various screen sizes (mobile, tablet, and desktop).  
- **Footer with Links:** Provides navigation to essential information and support pages.  

## Technologies Used  

- **HTML5:** For semantic structure.  
- **CSS3:** For styling and responsiveness.  
- **Google Fonts:** For enhancing typography.
